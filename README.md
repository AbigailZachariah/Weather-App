# 🌤️ Real-Time Weather Application

A clean, responsive weather application built with *HTML5, **CSS3, and **JavaScript. This project fetches live data from the **OpenWeatherMap API* to provide current weather conditions for any city worldwide.

## ✨ Features
* *Live Data:* Real-time temperature, humidity, and wind speed.
* *Dynamic Icons:* Weather icons change automatically based on conditions (Clouds, Clear, Rain, Drizzle, Mist, etc.).
* *Search Functionality:* Get weather details for any city by entering its name.

## 🛠️ Tech Stack
* *Frontend:* HTML, CSS, JavaScript
* *API:* [OpenWeatherMap API](https://openweathermap.org/api)
